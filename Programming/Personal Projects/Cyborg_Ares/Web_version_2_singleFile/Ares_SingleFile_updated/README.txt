Ares Single-File Website
------------------------
How to use:
- Open index.html in a browser.
- Click the nav buttons to switch the visible section.
- Inspect the bottom <script> to see how sections are toggled (no frameworks needed).

Learning suggestions:
- This is a small SPA-like pattern. Compare this file to the multi-page version to learn
  when to use separate files vs. a single-file app.
