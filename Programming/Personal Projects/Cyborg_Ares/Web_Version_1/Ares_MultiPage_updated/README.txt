Ares Multi-Page Website
-----------------------
How to use:
- Open each .html file in a browser from this folder (index.html is the entry page).
- 'assets' contains images and a copy of Cyborg.docx used as reference.
- Files contain inline HTML comments explaining structure and styling choices.

Learning suggestions:
- Compare structure.html and systems.html to see how content is separated across pages.
- Inspect the navigation in each HTML to learn how simple <a> links create a multi-page site.
